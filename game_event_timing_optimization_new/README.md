# 🎮게임 이벤트 시기 최적화(Game Event Timing Optimization)

1. 프로젝트 개요
   - 게임 내 이벤트 시기 최적화로 사용자 참여와 매출 극대화 목표
   - 데이터 분석과 시각화를 통해 최적의 이벤트 타겟과 시기를 제안
  
2. 데이터
   - 유저 일별 상태(user_info_daily_final)
   - 유저 재화 획득,소비 로그(gamemoney_incdec_daily_final)

3. 분석
   - 과제 1 : 탐색적 데이터 분석
   - 과제 2 : 강화 이벤트 시기 도출 분석
  
4. 사용 기술
   - Python : pandas, matplotlib, seaborn, statsmodels, ..
   - Tableau : 시각화
  
5. 프로젝트 구조
   - data : 원본 데이터
   - 게임 이벤트 시기 최적화.ipynb : 분석 코드 파일
   - 게임 이벤트 시기 최적화.pdf : 프로젝트 발표 자료
